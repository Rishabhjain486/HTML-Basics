Visibility

1. display: none; - makes the element go away completely. This comes in handy while making responsive websites where we may want to hide certain things when the website is access on a mobile device
2. visibility: hidden; - similar to display none property, but the difference here is that the space of the block stays there where as in display none the whole thing goes completely
3. So the display none is used more often than the visibility hidden property

Oder of precedence

The styling which is done later overrides the styling done above. But this is not the case always.

Overriding the order of precedence

We can override the order of precedence (or override another style called externally or something) by using the important flag
So even thought the style using the class order2 has a higher precedence, we have overridden it using the important flag
now no matter what else is there, the style with the important flag will get the precedence

Negative margins

These are rarely ever used
we can give a -ve value to the margin and the element will go in the opposite direction (ie outside the page)
we can give negative top bottom left etc values
this does not work on the padding, padding doesnt take -ve values
